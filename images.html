What Is an Image Tag?

The <img> tag embeds an image in a page. It does not need a closing tag, and it relies on attributes to work correctly.

The src Attribute

Tells the browser where to find the image file.

The alt Attribute

Describes the image in words, for screen readers and for cases when the image cannot load.
