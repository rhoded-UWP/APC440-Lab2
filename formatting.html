What Is Formatting?

Formatting tags change how text is presented, such as making it bold or italic, without using any CSS.

strong

This text is strong, meaning it is important.

em

This text is emphasized.

mark

This text is marked, like a highlighter.

small

This text is small.

del and ins

This text is deleted and this text is inserted.

sub and sup

Water is H2O. This is an example1 of a superscript.

blockquote

This is a quotation set apart from the rest of the text.

code and pre

Inline code looks like this: let x = 5;

function greet() {
    return "Hello, world!";
}
