What Is a Semantic Tag?

A semantic tag is an HTML element that describes the meaning of the content inside it, rather than just how it looks. Semantic tags make a page easier to read for both developers and screen readers.

header

Holds introductory content, such as a page title, near the top of the page or a section. This page uses one above.

nav

Holds the navigation links for a site. This page uses one above.

main

Holds the primary content of the page. This page's main content, including this list, is inside a <main> tag.

section

Groups related content together under a theme.

Example Section

This block of text is wrapped in a <section> tag.

article

Holds a self-contained piece of content, such as a blog post or news story.

Example Article

aside

Holds content that is related but separate from the main content, like a sidebar note.

This is an example of an <aside>.

footer

Holds closing content, such as copyright information. This page uses one below.
