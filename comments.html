HTML Comments

A comment lets a web designer leave notes in the source code that the browser never renders on the page. Comments start with <!-- and end with -->.

<!-- This is a comment, it will not display in the browser -->
<p>This paragraph is visible.</p>

This page actually contains a hidden comment right above this paragraph — view the page source (Ctrl+U in most browsers) to find it.

Why Use Comments?

  • Explaining why a section of markup exists
  • Leaving reminders or to-do notes for later
  • Temporarily disabling a block of HTML while testing, without deleting it
