What Is Web Accessibility?

Accessibility means designing a website so that people with visual, auditory, motor, or cognitive disabilities can still use it. Much of accessible design starts with writing good HTML, not with add-on tools.

Practices Covered on This Site

  • Using descriptive alt text on every image, as shown on the Images page
  • Using semantic elements like <nav> and <main>, covered on the Semantic Tags page, so screen readers can jump between regions of a page
  • Marking emphasis with meaningful tags such as <strong> and <em> instead of relying on visual style alone, as shown on the Formatting Text and Quotes page
  • Giving links clear, descriptive text rather than "click here"
  • Keeping enough color contrast between text and background
  • Making sure every interactive element can be reached and used with a keyboard
