The <img> Tag

Images are added to a page with the <img> tag. Unlike most HTML elements, it has no closing tag and no content between tags — it is a void element that just points to a file.

<img src="html5-logo.svg" alt="HTML5 logo" width="150">

Key Attributes

  • src — the path or URL to the image file (required)
  • alt — a text alternative shown if the image fails to load and read aloud by screen readers
  • width and height — set the display size

Why alt Matters

The alt attribute is not optional in good web design. It keeps a page usable for people using screen readers, on slow connections, or when an image link breaks. This is one of the core ideas covered on the Accessibility page of this site.
