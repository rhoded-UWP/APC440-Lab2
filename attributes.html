What Is an Attribute?

An attribute gives extra information about an HTML element. Attributes are written inside the opening tag as a name and a value, like name="value".

The href Attribute

Used on the <a> tag to set the destination of a link.

Example: Home

The src Attribute

Used on the <img> tag to point to an image file.

Example: <img src="logo.png" alt="Site logo">

The alt Attribute

Used on the <img> tag to describe an image for screen readers or if the image cannot load.

The id Attribute

Gives an element a unique name so it can be linked to or referenced elsewhere on the page.

The title Attribute

Adds a tooltip that appears when a user hovers over an element.

Example: Home
