What Are HTML Attributes?

Attributes give HTML elements extra information. They are always written in the opening tag as name="value" pairs. A single element can carry more than one attribute.

<a href="https://www.w3schools.com" title="Visit W3Schools">W3Schools</a>

Some of the most common attributes in web design:

  • href — the destination URL of a link
  • src — the file path for an image or other embedded resource
  • alt — a text description of an image, used for accessibility
  • title — extra information shown as a tooltip
  • id and class — hooks used to target elements with CSS or JavaScript

HTML Links

The anchor tag, <a>, is what turns plain HTML pages into a connected website. Its href attribute points to another page, another part of the same page, or an external site. Links can be relative (pointing to a file in the same project, like image.html) or absolute (a full web address starting with https://).
