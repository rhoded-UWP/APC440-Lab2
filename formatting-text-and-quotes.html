Text Formatting Elements

HTML has several inline elements for changing how text is displayed. Some are purely visual, and some also add meaning that assistive technology can announce.

  • <b> bold text (visual only): bold text
  • <strong> important text: strong text
  • <i> italic text (visual only): italic text
  • <em> emphasized text: emphasized text
  • <mark> highlighted text: marked text
  • <small> smaller text: small text
  • <del> deleted text: deleted text
  • <ins> inserted text: inserted text
  • <sub> and <sup>: H2O and E=mc2

Quotations

The <blockquote> element sets off a long, block-level quotation, while <q> is used for a short inline quote. The <cite> element names the source of a quote.
